# prismagram

instagram clone with Express + Prisma + React + React Native and NomadCoders

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [x] Like / Unlike a photo
- [x] Search by user
- [x] Search by location
- [x] Comment on a photo
- [x] Follow / Unfollow User
- [ ] See user profile
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit the photo (Delete)
- [ ] See the feed
